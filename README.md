# Chine Brickwork Commercial Platform

A self-contained single-page dashboard for Chine Brickwork combining:

- **Job Chasing CRM** — tracks submitted tenders, chase dates, and communication history
- **Marketing Agent** — monitors tender portals, contractor websites, and planning databases

## Usage

Open `index.html` in any modern browser (Chrome, Edge, Safari, Firefox). No installation or internet connection required.

## Deployment

Deployed via [Vercel](https://vercel.com). Any push to `main` automatically redeploys.
